> **Связанные документы:** [`BRIEF.md`](../BRIEF.md) §5, [`PLAN.md`](../PLAN.md), [`FORMULAR.md`](../FORMULAR.md).

## Как пользоваться

1. Редактируйте **«Согласованный вид»** в файле экрана.
2. Переносите в `bot.view.Texts` / `Keyboards`.
3. Статус: `согласовано` → `реализовано`.

## Ключевые решения (кратко)

- **ФИО первым**, ник/id в скобках — ссылка на badminton4u.
- **Рейтинги:** 👤 (S) и 👥 (D) — `MessageEmoji`.
- **Дисклеймер:** `/start` + `/help`, не на data-экранах.
- **S:** не в UI (только в P3).
- **H2H:** без разряда; «Изменить соперника»; пара vs пара — этап 5.1.
- **Поиск — кнопки:** `Фамилия Имя (город) 👤 … 👥 …` (без отчества; рейтинги только если есть).
- **Соперники:** только ФИО + `🟢W 🔴L 📊%` в `<pre>`.
