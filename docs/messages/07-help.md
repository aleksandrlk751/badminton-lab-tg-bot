# Справка (`/help`)

| Поле | Значение |
|---|---|
| **ID** | `help` |
| **Триггер** | `/help`, кнопка «ℹ️ Помощь» |
| **Статус** | `реализовано` |
| **Код** | `Texts.help()` |

## Согласованный вид

Эмодзи в пунктах — **как на кнопках меню**.

```
Справка

• 🔍 Найти игрока — введите фамилию или ник (от 3 символов)
• Карточка — рейтинги 👤/👥, последний турнир, 🤺 соперники
• 🆚 Сравнить (H2H) — сравнение двух игроков по всем встречам
• 📈 История рейтинга — график (скоро)

Данные: игроки Москвы и МО, глубина 3 года, обновление раз в сутки.
```

<b>Справка</b>

*(Исправить опечатку «Сcompareнить» → «Сравнить» при переносе в код.)*
