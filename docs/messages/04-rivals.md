# Соперники

| Поле | Значение |
|---|---|
| **ID** | `rivals` |
| **Триггер** | «🤺 Соперники» |
| **Статус** | `реализовано` |
| **Код** | `Texts.rivals()`, `Keyboards.rivals()` |

## Согласованный вид

```
Соперники · Иванов Иван · Все

1. Петров Пётр (Foe)     3–1 (75%)
2. Сидорова Анна (18499)  2–2 (50%)

Стр. 1/3 · всего 18
```

**Клавиатура:**

```
[ • Все ] [ MD ] [ WD ] [ XD ] [ D ]
[ ◀ ] [ 1/3 ] [ ▶ ]
[ ⬅️ К карточке              ]
```

## Правила

| Правило | Значение |
|---|---|
| Заголовок | `Соперники · {ФИО} · {фильтр}` |
| Строка | `{n}. {ФИО} ({nick/id})` + `W–L (win%)` |
| Город | **не показываем** |
| S (сыгранность) | **не показываем** |
| «Все» | W–L **суммируется** по D/MD/WD/XD на соперника |
| Footer | **нет** |

## Пустые состояния
