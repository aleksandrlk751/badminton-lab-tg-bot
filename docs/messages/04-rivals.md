# Соперники

| Поле | Значение |
|---|---|
| **ID** | `rivals` |
| **Триггер** | «🤺 Соперники» |
| **Статус** | `реализовано` |
| **Код** | `Texts.rivals()`, `Keyboards.rivals()`, `MessageEmoji` |

## Согласованный вид

```
Соперники · Иванов Иван · Все

1. Петров Пётр          🟢3  🔴1  📊75%
2. Сидорова Анна        🟢2  🔴2  📊50%

Стр. 1/3 · всего 18
```

Список строк — в `<pre>`: моноширинный шрифт Telegram, метрики 🟢/🔴/📊 выровнены в один столбец после самого длинного ФИО на странице.

**Клавиатура:**

```
[ • Все ] [ MD ] [ WD ] [ XD ] [ D ]
[ ◀ ] [ 1/3 ] [ ▶ ]
[ ⬅️ К карточке              ]
```

## Правила

| Правило | Значение |
|---|---|
| Заголовок | `Соперники · {ФИО} · {фильтр}` |
| Строка | `{n}. {ФИО}` + `🟢W  🔴L  📊%` (без ника в скобках) |
| Выравнивание | `<pre>`, pad до max длины префикса `{n}. {ФИО}` на странице |
| Город | **не показываем** |
| S (сыгранность) | **не показываем** |
| «Все» | W–L **суммируется** по D/MD/WD/XD на соперника |
| Footer | **нет** |

## Пустые состояния

| Ситуация | Поведение |
|---|---|
| Нет соперников | alert: `Соперников пока нет.` — карточку не менять |
| Нет в разряде | `Встреч в разряде MD нет.` + чипы |
