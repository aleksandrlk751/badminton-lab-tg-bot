## Scope MVP

- **Игрок vs игрок**, все встречи, **без выбора разряда**.
- **Пара vs пара** — этап **5.1** (отдельно).
- **S** — не показываем в UI (только внутри P3).

## Wizard (2 шага)

### Шаг 1 — только при `/h2h` или «🆚 Сравнить»

```
🆚 H2H · шаг 1/2

Введите фамилию или ник первого игрока:
```

→ нечёткий поиск; при нескольких — список кнопок.

### Шаг 2

```
🆚 H2H · шаг 2/2

Иванов Иван (Rocket)
Введите фамилию или ник второго игрока:
```

**С карточки:** шаг 1 пропускается (A = владелец карточки).

```

```

### «Изменить соперника»

Игрок A **фиксирован** → снова шаг 2 (ввод / выбор B) **новым сообщением**; экран предыдущего H2H остаётся в чате.
