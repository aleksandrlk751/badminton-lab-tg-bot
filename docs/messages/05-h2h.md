# Head-to-head (H2H)

| Поле | Значение |
|---|---|
| **ID** | `h2h` |
| **Статус** | `частично реализовано` — wizard и экран есть; **DoD — этап 7** |
| **Код** | `H2hFlowHandler`, `Texts.h2h*()`, `H2hService` |

## Scope (целевой DoD, этап 7)

- **Игрок vs игрок** после выбора **дисциплины** D / MD / WD / XD (как [`BRIEF.md`](../BRIEF.md) §5).
- **Тип пары** на экране — `PairCompositionService` по `player.sex` (этап 5); предупреждение при несоответствии разряду.
- **Пара vs пара** (4 игрока) — отдельный сценарий в рамках этапа 7.
- **S** — не показываем в UI (только внутри P3).

## Wizard

### Шаг 1 — только при `/h2h` или «🆚 Сравнить»

```
🆚 H2H · шаг 1/3

Введите фамилию или ник первого игрока:
```

### Шаг 2

```
🆚 H2H · шаг 2/3

Иванов Иван (Rocket)
Введите фамилию или ник второго игрока:
```

**С карточки:** шаг 1 пропускается (A = владелец карточки).

### Шаг 3 — выбор дисциплины (DoD этап 7)

Кнопки: **D**, **MD**, **WD**, **XD** (и при необходимости **S** для одиночных матчей — вторично).
