# Карточка игрока

| Поле | Значение |
|---|---|
| **ID** | `player-card` |
| **Триггер** | callback `card:{id}` |
| **Статус** | `реализовано` |
| **Код** | `Texts.card()`, `Keyboards.card()`, `MessageEmoji` |

## Согласованный вид

```
Иванов Иван (Rocket)
Москва

Рейтинги
👤  320
👥  535

Форма  👊  +1.2
Стабильность  🟢

Игровой акцент 📊
Микст
14 игр за полгода

Рекомендуемая категория 🧮
Женская пара

Последний турнир
Женская лига WDC
14.06.2026 · 2-е место
```

Примеры результата турнира:

| Данные | Строка |
|---|---|
| 2-е место | `14.06.2026 · 2-е место` |
| вылет в 1/4 | `14.06.2026 · вылет в 1/4` |
| группа | `14.06.2026 · не вышел из группы` |

**Клавиатура:**

```
[ 🤺 Соперники                 ]
[ 🆚 H2H        ] [ 📈 История рейтинга ]
[ ⬅️ В меню                    ]
```

## HTML (фрагмент)

```html
<b>{fullName}</b> (<a href="https://badminton4u.ru/players/{id}">{nick_or_id}</a>)
{city}

<b>Рейтинги</b>
👤  {rating_s}
👥  {rating_d}

Форма  👊  {form}
Стабильность  {stability_emoji}

<b>Игровой акцент</b> 📊
{preference_label_cap}
{preference_games} за полгода

<b>Рекомендуемая категория</b> 🧮
{strength_label_cap}

<b>Последний турнир</b>
{tournament_name}
{date} · {result_label}
```

## Правила

| Поле | Правило |
|---|---|
| Ник | ссылка; если пусто → `{id}` |
| Рейтинги | 👤 / 👥 с ненулевым значением; если нет ни одного — 🤷‍♂️ (`MessageEmoji.UNKNOWN`) |
| Форма | подпись «Форма», затем 👊 (`MessageEmoji.FORM`), формат `{sign}{one decimal}` (`+1.2`); при отсутствии данных — 🤷‍♂️ |
| Стабильность | подпись «Стабильность», затем emoji зоны (§2.8.1 `FORMULAR.md`): 🔴 `<70`, 🟡 `70–80`, ⚪ `80–86`, 🟢 `86–92`, 🔥 `92+`; **без числа**; при отсутствии данных — 🤷‍♂️ |
| Игровой акцент | заголовок с 📊; при достаточном весе — категория с заглавной буквы и число игр за полгода на отдельной строке; иначе 🤷‍♂️ |
| 📊 | тип пары (`Мужская пара` / `Женская пара` / `Микст`) и `{N} за полгода` на следующей строке (доля % — только в аналитике) |
| Рекомендуемая категория | заголовок с 🧮 и категория с заглавной буквы (ось 2); δ и винрейт — в аналитике; при отсутствии данных — 🤷‍♂️ |
| Последний турнир | название + дата · результат; при отсутствии — 🤷‍♂️ |
| Footer «Данные на…» | **нет** (дисклеймер в `/start` и `/help`) |
| 🤺 Соперники | только если `hasRivals` |

## Пустые состояния

| Ситуация | Текст |
|---|---|
| Игрок не найден | `Игрок не найден.` |
| Нет данных по полю | 🤷‍♂️ (`MessageEmoji.UNKNOWN`) в соответствующей секции карточки |
