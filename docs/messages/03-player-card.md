# Карточка игрока

| Поле | Значение |
|---|---|
| **ID** | `player-card` |
| **Триггер** | callback `card:{id}` |
| **Статус** | `реализовано` |
| **Код** | `Texts.card()`, `Keyboards.card()`, `MessageEmoji` |

## Согласованный вид

```
Иванов Иван (Rocket)
Москва

Рейтинги
👤  320
👥  535

👊  +1.2

Игровой акцент
🎯  микст · 58% · 14 игр за полгода
💪  женские пары · +2.3 · 🟢 78%

Последний турнир
Женская лига WDC
14.06.2026 · 2-е место
```

Примеры результата турнира:

| Данные | Строка |
|---|---|
| 2-е место | `14.06.2026 · 2-е место` |
| вылет в 1/4 | `14.06.2026 · вылет в 1/4` |
| группа | `14.06.2026 · не вышел из группы` |

**Клавиатура:**

```
[ 🤺 Соперники                 ]
[ 🆚 H2H        ] [ 📈 История рейтинга ]
```

## HTML (фрагмент)

```html
<b>{fullName}</b> (<a href="https://badminton4u.ru/players/{id}">{nick_or_id}</a>)
{city}

<b>Рейтинги</b>
👤  {rating_s}
👥  {rating_d}

👊  {form}

<b>Игровой акцент</b>
🎯  {preference_label} · {preference_share}% · {preference_games} за полгода
💪  {strength_label} · {strength_delta} · 🟢 {strength_win_rate}%

<b>Последний турнир</b>
{tournament_name}
{date} · {result_label}
```

## Правила

| Поле | Правило |
|---|---|
| Ник | ссылка; если пусто → `{id}` |
| Рейтинги | только 👤 / 👥 с ненулевым значением; число — обычный текст |
| Форма | 👊 (`MessageEmoji.FORM`), формат `{sign}{one decimal}` (`+1.2`); строка скрыта без матчей с дельтой |
| Игровой акцент | две строки: 🎯 предпочтение, 💪 сильная сторона; секция скрыта при недостаточном весе матчей |
| 🎯 | тип пары (`мужская пара` / `женская пара` / `микст`), доля %, число игр за полгода по доминирующему типу |
| 💪 | тип с лучшей взвешенной δ, `{sign}{one decimal}`, 🟢 + взвешенный винрейт % |
| Footer «Данные на…» | **нет** (дисклеймер в `/start` и `/help`) |
| 🤺 Соперники | только если `hasRivals` |

## Пустые состояния

| Ситуация | Текст |
|---|---|
| Игрок не найден | `Игрок не найден.` |
| Нет рейтингов | секция «Рейтинги» скрыта или «нет данных» |
| Нет матчей с дельтой | строка 👊 скрыта |
| Мало данных для акцента | секция «Игровой акцент» скрыта |
| Нет турниров | секция «Последний турнир» скрыта |
