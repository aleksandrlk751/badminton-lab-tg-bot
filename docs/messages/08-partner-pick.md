```

[ 📅 Ближайшие турниры     ]
[ 🔗 Поиск по ссылке       ]
```

```
Вставьте ссылку на турнир с badminton4u.ru …
```

## Список кандидатов (черновик)

```
Партнёры · Кубок LAB MD 500
Лимит пары: 500
Для: Иванов Иван (Rocket) · 🅳 380

Уже играли
  ⭐ Петров Пётр Сергеевич
  🅳 395 · ср. 388 · подходимость: хорошая (62%)

Новые кандидаты
  Сидоров Алексей
  🅳 370 · ср. 375 · подходимость: средняя (38%)

[ ⬅️ Назад                   ]
[ ⬅️ В меню                   ]
```

«Назад» — снова ввод ФИО/ника для **того же турнира** (другой игрок).

Строка: **ФИО** (если нет — ник), парный рейтинг кандидата, средний `(вы+кандидат)/2`, уровень подходимости по score 0–100. ⭐ — идеальный кандидат (история + категория + форма).

*(Footer «Данные на…» — не на экране; дисклеймер в `/help`.)*
