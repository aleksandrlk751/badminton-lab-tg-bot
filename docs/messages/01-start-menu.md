# Главное меню (`/start`)

| Поле | Значение |
|---|---|
| **ID** | `start-menu` |
| **Триггер** | `/start` |
| **Статус** | `реализовано` |
| **Код** | `Texts.menu()`, `Keyboards.mainMenu()` |

## Согласованный вид

```
Badminton LAB

Статистика игроков Москвы и МО: рейтинги, соперники, H2H.

Введите фамилию или ник — или выберите действие:

Данные по Москве и МО · обновление ежедневно
```

**Клавиатура:**

```
[ 🔍 Найти игрока              ]
[ 🆚 Сравнить (H2H)            ]
[ 🤝 Партнёр на турнир         ]
[ ℹ️ Помощь                    ]
```

## HTML

```html
<b>Badminton LAB</b>

Статистика игроков Москвы и МО: рейтинги, соперники, H2H.

Введите фамилию или ник — или выберите действие:

<i>Данные по Москве и МО · обновление ежедневно</i>
```

## Кнопки

| Кнопка | callback |
|---|---|
| 🔍 Найти игрока | `menu:search` |
| 🆚 Сравнить (H2H) | `menu:h2h` |
| 🤝 Партнёр на турнир | `menu:partner` |
| ℹ️ Помощь | `menu:help` → [`07-help.md`](07-help.md) |
