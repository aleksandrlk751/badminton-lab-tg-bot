## 2c. Результаты поиска

**Заголовок** (склонение `найден` / `найдено` / `игрок` / `игрока` / `игроков`):

```
Нашёл 3 игроков, подходящих под критерии поиска
Выберите из предложенных вариантов
```

| n | Заголовок |
|---|---|
| 1 | `Нашёл 1 игрока, подходящего под критерии поиска` |
| 2–4 | `Нашёл N игрока, подходящих под критерии поиска` |
| 5+ | `Нашёл N игроков, подходящих под критерии поиска` |

**Кнопки** (≤64 символа — лимит Telegram; при переполнении см. ниже):

```
Иванов Иван (Москва) 👤 320 👥 535
```

**Шаблон:**

```
{Фамилия Имя} ({город}) 👤 {rating_s} 👥 {rating_d}
```

| Часть | Правило |
|---|---|
| ФИ | **без отчества**; если пусто → **ник**; если и ника нет → `Игрок {id}` |
| Город | в скобках; если пусто — скобки **не выводим** |
| 👤 / 👥 | как на карточке: **только если рейтинг есть** |
| Ник | **не** в кнопке (акцент на фамилию) |

**Примеры:**

```
Иванов Иван (Москва) 👤 320 👥 535
Петров Пётр (Химки) 👥 412
Сидорова Анна 👤 280 👥 390          ← нет города — скобки не выводим
Rocket 👥 500                       ← нет ФИО в БД — показываем ник
```

**Скобки `(...)`** — **только город** (исключение из правила «ФИО + ссылка» — лимит кнопки). Id в кнопке **не используем**; id — текст ссылки в **сообщениях**, когда нет ника (см. [`00-principles.md`](00-principles.md)).

**Усечение** (лимит Telegram — **64 символа** на подпись кнопки).

Базовый формат: `{Фамилия Имя} ({город}) 👤 {rating_s} 👥 {rating_d}` (отчество **не выводим**).
Если не влезает — **по порядку**:

| Шаг | Действие | Было → стало |
|---|---|---|
| 1 | Убрать **город** (и скобки) | `(Москва)` → — |
| 2 | Убрать **👤 рейтинг** | `👤 320` → —; **👥 остаётся** |
| 3 | Обрезка «…» в конце | крайний случай |

**Пример цепочки** (игрок с отчеством в БД — на кнопке всё равно без него):

```
Иванов Иван (Москва) 👤 320 👥 535   ← базовый
Иванов Иван 👤 320 👥 535            ← шаг 1
Иванов Иван 👥 535                   ← шаг 2
```

*(Реализация: `SearchButtonLabel`, `MessageEmoji`, `Keyboards.searchResults()`.)*

## Поиск (техника)

Нечёткий поиск `PlayerSearchService` + `pg_trgm`: фамилия, имя, ник, от 3 символов, до 10 результатов.
H2H-wizard использует **тот же** поиск + список кнопок при нескольких совпадениях.
